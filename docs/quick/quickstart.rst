Quickstart
==========

Getting the latest Parsl is easy, but requires a few steps.


Installing
==========

1. Download Parsl::

    $ git clone https://github.com/swift-lang/swift-e-lab.git parsl

2. Install::

    $ cd parsl
    $ python3 setup.py install

3. Use Parsl!

Requirements
============

Parsl requires the following :

* Python 3.5+

For testing:

* nose
* coverage
