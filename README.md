# Parsl - Parallel Scripting Library

Parsl is a parallel scripting library that enable easy parallelism and workflow design.

## Requirements:

Python3.5+


## How to get started:

Install:

Python3.5 and Python3-pip

Install Jupyter:

````
pip3 install jupyter
````
